cara kerja program :
1. kita mengetikkan kata loginnya
2. kita menggetikan username yang ingin kita pilih
3. kita bs langsung menggunakan command seperti:
	A. list untuk melihat siapa saja user yang sudah login kecuali diri sendiri
	B. sendall untuk membroadcast ke semua user yang aktif pada 
	   formatnya sendall [spasi] [pesan]
	C. sendto untuk membuat private message (pm) dengan user tertentu 
	   formatnya sendto [spasi] username [spasi] pesan
4. kalau mau langgsung keluar tinggal tekan tombol Ctrl + C
